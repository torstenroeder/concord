Person Concordancer 1.3

This is a tool to find general identifiers for persons, just by providing some basic data.

Method

Basically, all given properties are compared to the entries in a large database. Each match adds some points to a total score.

Points are weighted by importance and probability for identification.

Hints

  • Names, places and descriptions may contain more than one word.
  • Any query that exceeds a certain limit of results will be ignored.

Examples

  • a request with many details
  • a request with only name and description
  • a request without a name

Results

  • Results will be returned in XML.
  • if there are more than results, the following results will be displayed with a score of or more
  • more than results will not be displayed
