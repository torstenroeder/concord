Person Concordancer 1.1

This is a tool to find general identifiers for persons, just by providing some basic data. This is version 1.1.

Parameters

You can submit a request by a combination of URL parameters.

  • n - principal name (can contain more than one word)
  • on - other names (can contain more than one word)
  • db - date of birth (ISO 8601)
  • pb - place of birth
  • dd - date of death (ISO 8601)
  • pd - place of death
  • d - description keywords

examples:

  • a request with many details
  • a request with only name and description

Method

Person data is compared in sets of (predefined) contexts. Basically, every information needs to be confirmed by a second information in the same context to be plausible. As contexts can overlap, it is possible that one information can be evaluated more than once.

Every person that matches a context gains points according to the relevance of the context. Points are accumulated over all context comparisons. Thus, the maximum score can differ, depending on the request.

Results

  • results will be returned in XML
  • if there are more than 5 results, the following 5 results will be only displayed with a score of 40 or more
  • more than 10 results will not be displayed

Scores

Concordance of name and other names+30
Concordance of all name parts+15
Concordance of name and year of birth/death+30
Concordance of name and description+20
Concordance of exact date and place of birth+15
Concordance of exact date and place of death+15
